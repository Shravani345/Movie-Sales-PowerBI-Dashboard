# Movie Sales Analysis Dashboard (Power BI)

## Project Overview
This project is an interactive Power BI dashboard that analyzes movie sales, ratings, and trends across genres, years, and directors.

## Tools Used
- Power BI
- Excel Dataset

## Key Insights
- Total Box Office Revenue by Year
- Total Movies Released per Year
- Top Directors by Revenue
- Average IMDB Ratings by Genre
- Genre-wise Revenue Distribution

## Features
- Interactive slicers for Genre, Year, and Director
- KPI cards for Total Movies, Revenue, Votes, and Runtime
- Trend charts and visual analytics
